Somos el grupo milanesas

Esta pagina se trata de como hacer unos sandwiches de milanesas, de varios ingredientes que puedes elegir, luego de crearla lo podremos ver en una lista de sandwiches, cada sandwich esta hecho por diferentes usuarios que se registraron en la pagina, lo podremos ver en la lista de usuarios cuantos son y cuantos han creado cada uno. Cada pagina que se encuentra en este repositorio esta escrita con HTML, todas las paginas estan decoradas con Bootstrap, y las paginas de crear sandwiches o mostrar la lista de usuarios funcionan con Javascript y Flask, la base de datos esta hecha con PostgreSQL.
